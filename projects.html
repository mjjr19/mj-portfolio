Master's Project • 2024

AI Personal Trainer App

A custom GPT model for personalized fitness coaching

📋 Project Overview

For my Master's project at the University of Washington Foster School of Business, I developed an AI-powered personal trainer application that leverages a custom-trained GPT model from OpenAI. The application provides users with personalized workout plans, nutrition advice, and progress tracking.

2
Weeks Development Time
93%
User Satisfaction

Main Features

⚙️ Technical Details

The project involved training a specialized GPT model using OpenAI's API to understand fitness-related queries and provide personalized responses.

Personalized Workouts

Custom algorithms create tailored workout plans based on user goals, fitness level, and available equipment.

Nutrition Guidance

AI-powered nutrition recommendations aligned with workout goals and dietary preferences.

Progress Tracking

Advanced analytics to monitor user progress and dynamically adjust recommendations.

🔧 Implementation Process

Week 1
Model Training & Development
Fine-tuned GPT model on fitness dataset and developed initial API integration.
Week 2
User Interface & Testing
Created user interface and conducted comprehensive testing with fitness enthusiasts.
Current
Refinement & Expansion
Ongoing refinement of recommendation algorithms and adding new features.

✅ Key Components

  • Custom GPT model fine-tuned on fitness and nutrition datasets
  • User profile creation and storage for personalized recommendations
  • Workout plan generation based on user goals and constraints
  • Nutrition advice tailored to individual needs
  • Progress tracking and adaptive recommendations

🏆 Outcomes & Impact

The application demonstrated high accuracy in generating personalized fitness plans and received positive feedback from test users. The project was completed within a two-week timeframe and showcased the practical applications of AI in the fitness industry.
